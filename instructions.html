1. How Can I Reveal A Hidden Element?

In this example we will reveal the Overlay that is shown on hover.

1. First, select the element you want to reveal. You can do this by finding the hidden element in the Navigator Panel.
2. Go to the Navigator and find the Overlay element.
3. Go to the Style Panel and change the Display Setting from None to Flex or Block.

2. How Can I Reveal The Top Dropdown Menu?

In this example we will reveal the Overlay that is shown on hover.

1. Locate the Top Dropdown Container and select it (click on it)
2. Go to the Style Panel and change its display setting to Display Block
3. Remove its Height
4. Done!

Don't forget to set again as Display None and set Height as 0px when done.

3. How Can Remove The Fade Out On Click Effect From The Navbar?

1. Select the link you want to remove the Fade Out Effect
2. Go to the Interactions Panel and change its setting from "Fade Out On Click" to "None"
3. Done!

4. How Can Edit The Testimonial Section?

In this example we will reveal the Overlay that is shown on hover.

1. First click anywhere on the slide section to select the slide.
2. Once highlighted, go to the Settings Panel.
3. Use the arrows to select the slide to edit.
4. Done!

Don't forget to set again as Display None and set Height as 0px when done.
